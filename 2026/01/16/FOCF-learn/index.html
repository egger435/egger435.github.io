йҖҡиҝҮдә’иЎҘж»ӨжіўиҺ·еҸ–е§ҝжҖҒи§’еәҰ

Posted on 2026-01-16 Edited on 2026-07-12 In жҺ§еҲ¶

вҖғвҖғжңҖиҝ‘еңЁеҒҡиҪ®и…ҝе°ҸиҪҰпјҢе…¶дёӯйңҖиҰҒйҖҡиҝҮйҷҖиһәд»ӘиҺ·еҸ–е°ҸиҪҰзҡ„е®һж—¶и§’еәҰеҖјгҖӮеӯҰд№ дәҶдёҖдёӢжңүе…ідә’иЎҘж»Өжіўзҡ„еҶ…е®№пјҢеңЁжӯӨи®°еҪ•дёҖдёӢгҖӮ

вҖғвҖғдә’иЎҘж»Өжіўзҡ„ж ёеҝғжҳҜиһҚеҗҲйҷҖиһәд»Әе’ҢеҠ йҖҹеәҰи®Ўзҡ„ж•°жҚ®пјҢз”ЁйҷҖиһәд»ӘдҝқиҜҒеҠЁжҖҒе“Қеә”пјҢз”ЁеҠ йҖҹеәҰи®Ўдҝ®жӯЈй•ҝжңҹжјӮз§»пјҢиҫ“еҮәзЁіе®ҡдё”е®һж—¶зҡ„и§’еәҰдј°и®ЎгҖӮ

вҖғвҖғйҷҖиһәд»ӘжЈҖжөӢзі»з»ҹзҡ„дёүиҪҙи§’йҖҹеәҰеҖјпјҢеҠ йҖҹеәҰи®ЎжЈҖжөӢдёүиҪҙзҡ„еҠ йҖҹеәҰеҖјпјҲеңЁйқҷжӯўзҠ¶жҖҒдёӢпјҢдё»иҰҒжЈҖжөӢйҮҚеҠӣеҠ йҖҹеәҰзҡ„дёүиҪҙеҲҶйҮҸпјүгҖӮж №жҚ®д»ҘдёҠдј ж„ҹеҷЁж•°жҚ®пјҢжҲ‘д»¬йңҖиҰҒеҫ—еҲ°зі»з»ҹзҡ„е®һж—¶е§ҝжҖҒи§’еәҰжңүдёӨз§ҚеҠһжі•пјҡ1. йҖҡиҝҮйҷҖиһәд»Әз»ҷеҮәзҡ„и§’йҖҹеәҰж•°жҚ®з§ҜеҲҶеҫ—еҲ°и§’еәҰеҖјпјӣ2. йҖҡиҝҮеҠ йҖҹеәҰи®Ўз»ҷеҮәзҡ„еҠ йҖҹеәҰж•°жҚ®иҝӣиЎҢеҸҚжӯЈеҲҮиҝҗз®—еҫ—еҲ°и§’еәҰеҖјгҖӮ

1. и§’йҖҹеәҰз§ҜеҲҶеҫ—еҲ°и§’еәҰеҖј

ж–№жі•

вҖғвҖғжҲ‘д»¬зҹҘйҒ“пјҢеңЁжһҒе°Ҹзҡ„дёҖж®өж—¶й—ҙ$\Delta t$еҶ…еҸҜд»Ҙе°Ҷзі»з»ҹзҡ„иҝҗеҠЁи§ҶдёәеҢҖйҖҹгҖӮеӣ жӯӨиҝҷж®өж—¶й—ҙеҶ…иҪ¬иҝҮзҡ„и§’еәҰдёә$\Delta \theta = \omega \Delta t$гҖӮеҜ№е…¶иҝӣиЎҢз§ҜеҲҶе°ұеҸҜд»ҘеҫҲе®№жҳ“еҫ—еҲ°еҪ“еүҚзҡ„и§’еәҰеҖјпјҡ

$$\theta_{gyro} = \int_0^t\omega dt$$

еұҖйҷҗжҖ§

вҖғвҖғдҪҶжҳҜиҝҷз§Қж–№жі•еӯҳеңЁдёҖдёӘй—®йўҳпјҢе°ұжҳҜд»ҺйҷҖиһәд»ӘиҺ·еҸ–зҡ„и§’йҖҹеәҰж•°жҚ®жҜҸж—¶жҜҸеҲ»йғҪеӯҳеңЁиҜҜе·®пјҢиҖҢз§ҜеҲҶиҝҷз§Қж“ҚдҪңдјҡе°Ҷиҝҷз§ҚиҜҜе·®зҙҜз§ҜеҜјиҮҙз§ҜеҲҶеҫ—еҲ°зҡ„и§’еәҰеҖјеӯҳеңЁжјӮз§»гҖӮиҖҢиҝҷз§ҚжјӮз§»иҜҜе·®еңЁдҪҺйў‘ж®өдјҡиў«з§ҜеҲҶж”ҫеӨ§пјҢжңҖз»Ҳдё»еҜјиҫ“еҮәз»“жһңпјҢдҪҝеҫ—и§’еәҰдёҘйҮҚеҒҸзҰ»зңҹе®һеҖјгҖӮиҖҢеңЁй«ҳйў‘еңәжҷҜдёӢпјҢдҫӢеҰӮе°ҸиҪҰеҝ«йҖҹжҷғеҠЁпјҢжӯӨж—¶йҷҖиһәд»Әзҡ„жңүж•Ҳи§’йҖҹеәҰдҝЎеҸ·иҝңеӨ§дәҺдҪҺйў‘жјӮз§»зҡ„дҝЎеҸ·пјҢжӯӨж—¶еҜ№жңҖз»Ҳи§’еәҰзҡ„еҪұе“ҚеҸҜд»ҘеҝҪз•ҘгҖӮ

вҖғвҖғжүҖд»ҘжҖ»з»“дёҖдёӢпјҢйҖҡиҝҮи§’йҖҹеәҰз§ҜеҲҶеҫ—еҲ°зҡ„и§’еәҰеҖјеңЁеҠЁжҖҒжғ…еҶөдёӢиЎЁзҺ°е°ҡеҸҜпјҢиҖҢеңЁйқҷжҖҒжғ…еҶөдёӢиЎЁзҺ°ж¬ дҪігҖӮ

2. еҠ йҖҹеәҰж•°жҚ®еҸҚжӯЈеҲҮеҫ—еҲ°и§’еәҰеҖј

ж–№жі•

вҖғвҖғеҒҮе®ҡеңЁйқҷжӯўзҠ¶жҖҒдёӢи®©зі»з»ҹз»•yиҪҙж—ӢиҪ¬дёҖдёӘи§’еәҰ$\theta$пјҢж №жҚ®д»ҘдёҠеҲҶжһҗеҸҜд»ҘеҫҲе®№жҳ“еҫ—еҮәзі»з»ҹзҡ„дҝҜд»°и§’пјҡ

$$\theta_{acc} = \arctan \frac{ax}{az}$$

еұҖйҷҗжҖ§

вҖғвҖғдҪҶжҳҜиҝҷз§Қж–№жі•еҗҢж ·д№ҹеӯҳеңЁдёҖдёӘй—®йўҳпјҢе°ұжҳҜй«ҳйў‘ж®өзҡ„жҢҜеҠЁе№Іжү°е’ҢиҝҗеҠЁеҠ йҖҹеәҰдјҡжҺ©зӣ–жңүз”Ёзҡ„йҮҚеҠӣдҝЎеҸ·пјҢж— жі•еҮҶзЎ®иҺ·еҸ–йқҷжҖҒи§’еәҰеҸӮиҖғгҖӮжҜ”еҰӮеҝ«йҖҹжҷғеҠЁе°ҸиҪҰпјҢеҠ йҖҹеәҰи®Ўзҡ„иҫ“еҮәжҳҜйҮҚеҠӣеҠ йҖҹеәҰгҖҒй«ҳйў‘иҝҗеҠЁеҠ йҖҹеәҰзҡ„еҸ еҠ пјҢжӯӨж—¶жңүз”Ёзҡ„йҮҚеҠӣдҝЎеҸ·иў«й«ҳйў‘е№Іжү°ж·№жІЎпјҢж— жі•йҖҡиҝҮеҠ йҖҹеәҰи®Ўи®Ўз®—еҮәеҮҶзЎ®зҡ„и§’еәҰгҖӮ

вҖғвҖғжҖ»з»“дёҖдёӢпјҢе°ұжҳҜйҖҡиҝҮеҠ йҖҹеәҰеҸҚжӯЈеҲҮеҫ—еҲ°и§’еәҰеҖјеңЁйқҷжҖҒжғ…еҶөдёӢиЎЁзҺ°е°ҡеҸҜпјҢиҖҢеңЁеҠЁжҖҒжғ…еҶөдёӢиЎЁзҺ°ж¬ дҪігҖӮ

3. йҖҡиҝҮдә’иЎҘж»Өжіўдҝ®жӯЈи§’еәҰеҖј

вҖғвҖғйҖҡиҝҮд»ҘдёҠеҲҶжһҗпјҢжҲ‘д»¬еҸҜд»ҘеҸ‘зҺ°йҷҖиһәд»Әе’ҢеҠ йҖҹеәҰи®Ўзҡ„йў‘зҺҮзү№жҖ§жҳҜдә’иЎҘзҡ„гҖӮеӣ жӯӨжҲ‘д»¬еҸҜд»ҘеҜ№е…¶еҸ–й•ҝиЎҘзҹӯпјҢеңЁеҠЁжҖҒжқЎд»¶дёӢжӣҙзӣёдҝЎйҷҖиһәд»Әзҡ„ж•°жҚ®пјҢеңЁйқҷжҖҒжқЎд»¶дёӢжӣҙзӣёдҝЎеҠ йҖҹеәҰи®Ўзҡ„ж•°жҚ®гҖӮ

еҺҹзҗҶ

вҖғвҖғйҖҡиҝҮи®ҫе®ҡжқғйҮҚеҸӮж•°$K$жқҘйҖүжӢ©жҲ‘д»¬жҳҜжӣҙзӣёдҝЎйҷҖиһәд»Әзҡ„ж•°жҚ®иҝҳжҳҜжӣҙзӣёдҝЎеҠ йҖҹеәҰи®Ўзҡ„ж•°жҚ®пјҢжүҖд»ҘжңҖеҗҺзҡ„и§’еәҰиҫ“еҮәдёәпјҡ

$$\theta_{out} = K * \theta_{gyro} + (1-K) * \theta_{acc}$$

вҖғвҖғжқғйҮҚеҸӮж•°$K$зҡ„е…·дҪ“йҖүжӢ©йңҖиҰҒж №жҚ®зі»з»ҹзҡ„е…·дҪ“еә”з”ЁеңәжҷҜжқҘеҲҶжһҗгҖӮ

е®һзҺ°

е…·дҪ“е®һзҺ°д»Јз ҒеҰӮдёӢпјҡ

/*
  дёҖйҳ¶дә’иЎҘж»ӨжіўпјҢж ҮеҮҶзүҲжң¬
  gyro_    йҷҖиһәд»Әзү©зҗҶж•°жҚ®    В°/s
  acc_     еҠ йҖҹеәҰи®Ўзү©зҗҶж•°жҚ®  m/s^2
  filter_value->filtering_angle еӯҳж”ҫж»ӨжіўеҗҺзҡ„и§’еәҰж•°жҚ®пјҢдёәзңҹе®һзү©зҗҶйҮҸпјҢеҚ•дҪҚдёәи§’еәҰеҲ¶
*/
void FOCF_v2(float gyro_, float acc_, FOCF_STRUCT *filter_value)
{
    float angle_gyro;    // з”ұйҷҖиһәд»Әи®Ўз®—еҫ—еҲ°зҡ„и§’еәҰ
    float angle_acc_rad; // з”ұеҠ йҖҹеәҰи®Ўи®Ўз®—еҫ—еҲ°зҡ„и§’еәҰпјҲеј§еәҰеҲ¶пјү
    float angle_acc;     // з”ұеҠ йҖҹеәҰи®Ўи®Ўз®—еҫ—еҲ°зҡ„и§’еәҰпјҲи§’еәҰеҲ¶пјү
    
    angle_gyro = filter_value->angle_temp + gyro_ * filter_value->call_cycle;    // з§ҜеҲҶеҫ—еҲ°йҷҖиһәд»Әи§’еәҰ
    filter_value->angle_temp = angle_gyro;
    
    angle_acc_rad = atanf(acc_ / acc_z_);
    angle_acc = to_deg(angle_acc_rad);      // еҸҚжӯЈеҲҮеҫ—еҲ°еҠ йҖҹеәҰи®Ўи§’еәҰе№¶иҪ¬еҢ–дёәи§’еәҰеҲ¶
    
    // еҠ жқғжұӮе№іеқҮе№¶дҝ®жӯЈйӣ¶зӮ№
    filter_value->filtering_angle = filter_value->v2_K * angle_gyro + (1 - filter_value->v2_K) * angle_acc + filter_value->mechanical_zero;
}
